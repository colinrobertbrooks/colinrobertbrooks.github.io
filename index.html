Udacity Frogger

This is a clone of Frogger built from Udacity's frontend-nanodegree-arcade-game directory.

To play, access this link on a laptop or a desktop (mobile not currently supported).

Select an avatar, a dificulty rating and a duration, then click Start Game. Once the game starts, use the arrow keys on your keyboard to collect as many gems as you can within the time limit. Don't get hit by any bugs!
